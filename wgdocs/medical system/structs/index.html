DISEASE
Table / Enumeration

This is a table OR an enumeration.
WG standards dictate that enumerations that are not single use are to be in tables.

 View source »

The object structure for a disease. A disease MUST contain all of this data.

Fields

  • arg uniqueID type string

    The UniqueID of this disease.

  • arg name type string

    The Pretty Name of the disease, this is what displays to the client.

  • arg description type string

    A description of the disease.

  • arg character type number

    The Character who has this disease's ID.

  • arg time type number

    The os.time() this disease was applied at.

  • arg inflictor type number default: 0

    The Character who infected this person. When disease request a reference to an inflictor, 0 is what the response should be so that NOT operators can work.

MedicalData
Table / Enumeration

This is a table OR an enumeration.
WG standards dictate that enumerations that are not single use are to be in tables.

 View source »

The structure of Medical Data that's stored in ix.medical.characters.

It's important to note, that all characters should have a field added to them that points to their MedicalData.

Fields

  • arg wounds type table

    Any Wounds this character may have.

  • arg diseases type table

    Any Diseases this character may have.

OutbreakStruct
Table / Enumeration

This is a table OR an enumeration.
WG standards dictate that enumerations that are not single use are to be in tables.

 View source »

Outbreak Structure If I remember right, there is an outbreak system with infections or something, if I'm wrong woops.

Fields

  • arg characters type table

    The characters involved in this outbreak, indexed by Character ID, includes previously infected.

  • arg infections type table

    An ordered table (in order of time of infection) that contains a table with the following entries: { CharID, Time, Infector, Disease Instance }

  • arg patientZero type number

    The originator of this outbreak's Character ID.

  • arg startTime type number

    The time this outbreak started.
