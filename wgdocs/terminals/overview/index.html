Sx

Fields

  • arg NONE type number

    SafeNet will do nothing if this message errors or is rejected.

  • arg NOTIFY type number

    SafeNet will notify admins, and put a message in the log if this message errors or is rejected.

  • arg BREAK type number

    SafeNet will break the whole stack, and halt operation. The difference between this, and normal operation, is that SafeNet will not error messages, but merely log it and return it, this will not break the stack, so if there is a statement afterwards, it may work still, provided it didn't neec correct parameters.

    They are separated because not stopping exploits makes them way easier to find.

  • arg LOCK type number

    SafeNet will LOCK the Network Message from being run by the client again, until an Admin removes that lock.
