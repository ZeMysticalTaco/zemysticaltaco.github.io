HOOKS:PlayerCanUseTerminal(client, ent)
Function

Rough Documentation

This method, function or library is ROUGHLY DOCUMENTED!

This means that the documentation could potentially be incorrect.

Controls whether or not players can use the terminal

Returning anything but nil will cancel the operation.

Returning a string will cancel the operation, and show the specified text on the terminal.

Parameters

  • varclient type Player

    The Player trying o use the Terminal.

  • varent

    The Terminal they are trying to use.
