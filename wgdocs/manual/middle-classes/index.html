Middle Classes

Helix comes with the middleclass library, for objects that need to be instanced, we recommend using this library, Lua isn't really a good "Object Oriented" language, with the closest thing we have being MetaTables. The Middleclass library makes creating objects and instances of objects easy, and has inheritance.

Declaring an object is as easy as:

 local objectMeta = ix.meta.objectMeta or ix.middleclass( 'objectMeta' )
 AccessorFunc( objectMeta, 'foo', 'AccessorFuncDemonstration' ) --You can even use AccessorFunc on middleclass objects.

 -- Constructor function, this is called whenever a new object is initialized with meta:New(...)
 function objectMeta:Initialize( argumentone, argument_two, etc )
     self.foo = 'bar'
     self.uno = argumentone
 end

 --Meta Function
 function objectMeta:ReportStatus( )
     return 'I am a pretty object!'
 end

 -- Return a field.
 function objectMeta:GetSuperArgument( )
     return self.uno
 end

 -- Initializer.
 local object = objectMeta:New( 'This is an argument to the constructor!' )
 local msg = [[
         I am an object, whenever I am printed, I identify as: %s
         I am an instance of [object], with my own thoughts and feelings. I identify as: %s
         I am the status of [instance of object], here's what I have to say: %s
         I am the SuperArgument of [instance of object], here's what I am: %s
         And finally, the news.
     ]]
 MsgC( Format( msg, objectMeta, object, object:ReportStatus( ), object:GetSuperArgument( ), object:GetAccessorFuncDemonstration( ) ) )
