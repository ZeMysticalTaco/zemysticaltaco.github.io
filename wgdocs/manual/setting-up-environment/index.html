Setting up your Environment

Before you begin to work on your component, you're obviously going to need to know how to piece everything together, that's what this guide is for. Please note all code must follow the Code Standards. This is so that everyone can work together easily, please adapt your standards for this project to these standards, alternatively, we have a preset glualint configuration that will format it automatically for you.

If you are unaware of the development standards, please go read them here.

NOTE: This is only for vscode right now.

Server Content

Installing Helix

If you have never installed Helix before Read This Guide Here. You should not be relying on the Development Server to build, as that's for mass testing. Use your local copy.
